
# Aurelia — Luxury Watch Store ⌚

Aurelia is a modern, classy e-commerce platform for premium watches.  
Inspired by Swiss luxury (like Tissot), it combines **minimal UI, elegant typography, and smooth 3D animations** (powered by [Spline](https://spline.design)).

🔗 **Live Demo:** [Aurelia Watch Store](https://aurelia-beige.vercel.app/)
<img width="1440" height="742" alt="image" src="https://github.com/user-attachments/assets/9b019eaa-cd69-497a-9008-b41f487f3347" />

## ✨ Features

- 🎨 Elegant, dark-themed UI with luxury-inspired branding  
- ⌚ Product catalog with filters (brand, movement, strap, price, etc.)  
- 🛒 Fully functional cart and checkout flow (mock/demo setup)  
- 📱 Responsive design for all screen sizes  
- 🖼️ Spline 3D animation in hero section  
- 📰 Journal (blog) section for guides and updates  
- 📂 CMS-ready structure (Products, Collections, Journal)  

---

## 🚀 Tech Stack

- **Frontend:** React + Next.js  
- **Styling:** Tailwind CSS  
- **3D Animation:** Spline embed  
- **Deployment:** Vercel

## ⚙️ Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/your-username/aurelia-watch-store.git
cd aurelia-watch-store
npm install
npm run dev
````

Then open [http://localhost:3000](http://localhost:3000) in your browser.
